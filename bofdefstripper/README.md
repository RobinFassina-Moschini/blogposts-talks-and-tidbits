## BOFStripper python script
This script is made for when you are like me and PoC your BOFs in C before porting them to BOFs. 
I even go as far as to doing the development of my PoC on a Windows machine but writing my BOFs in a Linux machine. 
Obviously, importing a huge header file is nbd, because the compiler/linker will just take what it needs, but I like to have a nice overview of what functiondefs my programs use.
Therfore I create this small script - only been tested on two projects so far, which is why I am releasing it here for fun and hopefuly more sample size in case people wanna contribute. 

It works as follows, you take a huge bofdefs file - I like the one from the TrustedSec repo - https://github.com/trustedsec/CS-Situational-Awareness-BOF/blob/master/src/common/bofdefs.h
and you take your C source code and feed it to the script. 
The script will do some regex magic to find the API calls in bofdefs and match them with your source code and generate a bofdefs header file that includes any found bofdefs that match up with your C code.
This also obviously means that if the function is not defined in your giant bofdef template file, it will not be generated for you either.
In case that happens, feel free to PR your bofdef file with added function defs!
